## Nome do Projeto

EcoSolo – Agro Forte, Futuro Sustentável

## Objetivo

O EcoSolo é uma plataforma educacional desenvolvida para estudantes de Agronomia, produtores rurais e demais interessados na área agrícola. Seu objetivo é promover o conhecimento sobre conservação do solo, agricultura sustentável, uso eficiente dos recursos naturais e agricultura de precisão, incentivando práticas responsáveis que conciliem produtividade e preservação ambiental.

Além disso, o sistema permite que os usuários testem seus conhecimentos, realizem avaliações de sustentabilidade e compartilhem experiências relacionadas ao meio rural.

## Recursos Técnicos

O projeto oferece os seguintes recursos:

Página inicial com apresentação do projeto.
Teste de conhecimento sobre sustentabilidade agrícola.
Cálculo do Índice de Sustentabilidade do Solo (ISS).
Calculadora de Impacto Sustentável para propriedades rurais.
Biblioteca de textos educativos sobre agricultura sustentável.
Espaço colaborativo para compartilhamento de experiências e dúvidas.
Classificação automática dos resultados obtidos pelos usuários.
Interface responsiva para diferentes tamanhos de tela.
Hub de Acessibilidade

O EcoSolo foi desenvolvido visando facilitar a navegação e o acesso às informações por diferentes usuários, contando com:

Layout responsivo para computadores, tablets e celulares.
Estrutura organizada com menus de navegação simples.
Uso de cores contrastantes para melhorar a leitura.
Textos claros e objetivos.
Botões e formulários de fácil identificação.
Hierarquia visual através de títulos, subtítulos e cartões informativos.

Verde: sustentabilidade e meio ambiente.

Bege: naturalidade e conforto visual.

Experiência do Usuário (UX)

Feedback imediato após testes e cálculos.
Fácil acesso às principais funcionalidades.
Interface do Usuário (UI)
Design moderno e limpo.

Botões interativos com efeitos visuais.
Layout adaptável para dispositivos móveis.
